# fastCP

This respository houses the code to implement the greedy, label-switching algorithm for Core-Periphery (CP) detection from "Label-switching algorithm for fast core-periphery identification" by Eric Yanchenko and Srijan Sengupta. Please cite the paper if you use these codes!

sims_functions.R
Main functions, including Algorithm 1.

cp_be.py
Python code to run optimization algorithm from cpnet Python package

sims_sbm.R
Code to replicated stochastic block model (SBM) simulations from the paper.

sims_dcbm.R
Code to replicated degree-corrected block model (DCBM) simulations from the paper.
